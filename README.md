# Artifactiq Website

Marketing website for [Artifactiq](https://artifactiq.ai) - Intelligence in Every Frame.

## Tech Stack

- **Framework**: Next.js 14 (App Router)
- **Styling**: Tailwind CSS
- **Deployment**: Cloudflare Pages
- **Domain**: artifactiq.ai

## Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Deploy to Cloudflare Pages
npm run deploy
```

## Structure

```
website/
├── src/
│   └── app/
│       ├── page.tsx        # Landing page
│       ├── layout.tsx      # Root layout
│       └── globals.css     # Global styles
├── public/                 # Static assets
└── package.json
```

## Deployment

Deployed automatically via Cloudflare Pages on push to `main` branch.
